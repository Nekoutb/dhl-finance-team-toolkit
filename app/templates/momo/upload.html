How it works

  1. Upload the report — the tool reads the transaction table and the report header (kept for the PDF, not repeated on screen).
  2. Tick one transaction or a whole group (select-all available).
  3. Confirm or type the third party's name and account number — remembered for next time ({{ saved_count }} saved so far).
  4. Extract the PDF — provider logo + report header + the selected line(s); the file name starts with the transaction reference (e.g. C00598) — or send it by email directly.
