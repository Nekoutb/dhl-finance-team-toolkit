📬 Operator mailbox (IRO returns)

The dedicated inbox the tool reads to ingest operator payment references sent by email (subject PAYREF <account>). Statements and links are sent with the SMTP settings above; this section is only for RECEIVING the returns.

Only mail from a registered operator email is processed; everything else is quarantined on the Operator statements page. New mail is picked up every few minutes (and on “Fetch mail now”).

📲 MyDHLPay (scan-to-pay)

The public payment page at /pay dials Orange Money with these settings. The amount is filled per airwaybill automatically.

Prove it works

Office 365: host smtp.office365.com, port 587, TLS on — if your tenant blocks basic SMTP auth, IT can enable "Authenticated SMTP" for the mailbox or provide a relay. Gmail: host smtp.gmail.com, port 587, TLS on, and an app password (not your normal one) from myaccount.google.com → Security → 2-Step Verification → App passwords. Yahoo: host smtp.mail.yahoo.com, port 465 (SSL — supported) or 587, and an app password generated at login.yahoo.com → Account Security → Generate app password; your normal Yahoo password will be rejected.
