Prove it works

Office 365: host smtp.office365.com, port 587, TLS on — if your tenant blocks basic SMTP auth, IT can enable "Authenticated SMTP" for the mailbox or provide a relay. Gmail: host smtp.gmail.com, port 587, TLS on, and an app password (not your normal one) from myaccount.google.com → Security → 2-Step Verification → App passwords. Yahoo: host smtp.mail.yahoo.com, port 465 (SSL — supported) or 587, and an app password generated at login.yahoo.com → Account Security → Generate app password; your normal Yahoo password will be rejected.
